# 1.研究的本质：

研究是为了更好

你从别人那里学习了东西，如果超越了他，那就是研究，比他更好了

# 2.研究和学习的区别：

学习呈现的三个特征时练习，跟随，重复

可研究没有这三个特征

# 3.训练有效研究必备的三种意识

（1）作品意识：一个人在研究之前必须有强烈的作品意识，例如烹饪，你和你的家人都会烹饪，并且烹饪技术向他们学习的，但是他们做的饭只是为了填饱肚子，你做的饭是为了把每道菜都当做一个作品，这时候你就会发现你会研究这个烹饪，会越做越好，最后会出一套烹饪教程，这就是作品意识带来的研究驱动。作品不一定要很大，可能仅仅是一个细节就可以了。研究目标的时候不要总想着立即改变世界，而是首先激发你对研究事物的作品意识，这个最为关键。

你平时做事的时候给自己一句暗示，告诉自己我要从事这件事了，不是做这件事，而是从事，仅凭这一个词的变化，你的仪式感和作品意识就会被唤起。

（2）目标意识：防止研究过程跑偏

光有研究精神和作品意识，一个人很容易还会在研究方向上跑偏，就是因为缺乏清晰地目标意识

很多人在写论文的时候，刚写了几行字喝了两口茶，结果就突然打开维基百科认真的查看茶叶的起源，分类和制作工艺，看了两个多小时，这是很多人做研究的时候比较常见的分心状态，甚至已经有了拖延症的嫌疑了。

在研究过程中时刻盯住自己初始研究目标，时刻问自己，正在做的事会不会影响研究的主线，用一种上帝视角俯视整个事情的全貌，才能守住目标不跑偏。

（3）局外意识：

一定要清楚地认识到研究是需要付出成本的。

# 4.搜集资料：

要点：足量和节制

主动性能让你达到足量，而宽泛性能让你评判足量要足到什么程度。

宽泛性是有目标的多：

什么是宽泛性？我给它的具体描述是跳出单一的维度，带有目标意识的多

比如一位厨师，如果要学习做好一个菜品，他可能要反复练习炒同一道菜，因为厨师是学习者，但一位更偏向研究者的美食家要品鉴一道菜，他不是要重复吃这一个菜，甚至不能只吃这一个菜系，他需要的是有意识的广泛的品尝更多的菜，甚至要吃遍这世上所有的美食，这就是我说的跳出单一维度带有目标意识的多。

这个节制跟足量绝对不是矛盾的，人的时间精力是有限的，只有我们在搜集资料足量的基础上做到节制，才能让我们更有针对性的进行研究和完成研究。

# 这个节制分为三种情况：

1.避免过度沉溺于文本

过度沉溺的很容易跑偏题

2.对人的研究要去粉丝化

3.要避免情绪化

# 5.怎么用批判性思维去处理资料，要把握两个方向：一个是抓本质，一个是找规律

抓本质就是找不同，要找不同就需要做对比

找规律则是求相似，要求相似就需要做类比

跟学习相比，研究它不包含练习，研究需要主动，研究的文本更加宽泛，研究仅仅对文本进行剖析，而不用去掌握文本。这就是对研究本质的研究了，找到一个跟研究在感觉上类似的对象，时刻提出疑问，他们有什么不同，从多个角度多个维度进行细致的比较。

怎么探究事物的本质？答案是给你的研究对象树立一个相似对象或者竞争对手，去找研究对象和类似对象的不同

比如说我们研究电影，很多人喜欢看电影，觉得自己是个影迷，甚至有人声称自己年纪轻轻就看过三千部电影，但是就算看过三千部电影，你要让他说出电影和其他艺术表达形式的本质区别，他又说不出来。当你问他们什么是电影，甚至一部好电影好在哪里的时候，他们也解释不了。

6.你掌握了规律，你研究相关事物的时候借用这个规律就会容易得多，毕竟都是做图片处理的软件，其实有很多大型软件都有对应的傻瓜型软件，他们是相似的，你要掌握一个大型软件可以先从跟他对应的傻瓜型软件入手，效率就会更高。

总结：搜集完足够的资料后，对这些资料处理的关键有两条，做对比抓本质，已经做类比找规律

摸清一样事物的规律之后，你还能用这种规律去研究别的相关事务会事半功倍。

# 6.用写作促进你的研究

你怀着某个目的做研究，可能得到三种结果：

1.研究证明你原先的想法错了

2.研究证明你原先的想法对了，以上这两种，都不需要将研究结论成果化

3.研究让你有了新的发现，新的收货，这样的研究结论不但对你自己有帮助，而且对别人也可能会有很大的价值，我们就应该把研究结论成果化。

结论成果化最好的表达形式就是写作

要做到系统性，你的储备信息量要足够的大，这就是我们前面说的，信息搜索要足量，在一个相对完整的系统中提供足够的信息量，是学术性的保障，如果你的研究不成系统，信息达不到这个量，你只是只言片语零散的给出少量信息的话，那你对这个问题回答的底气也弱了，你可能自己心里都会打鼓，有没有什么你没有涉猎的信息？

我在网上看到一个很有趣的问题，有人问：李白喝的都是什么酒，被点赞最多的回答，信息量大到什么程度，从李白现存的九百多首诗中找到提及酒的有两百多首左右，于是给出了李白喝的酒有白酒，清酒，葡萄酒，鲁酒，菊花酒，桂酒等等，这就叫信息量大。

# 7.好文章要有足够的文本支撑

学术性需要文本支持，在研究的过程中当然要有自己的判断，但这种判断绝对不能是凭想象。生活中，跟别人谈论问题的时候不提供文本，仅仅表达自己的观点或者想法，但是就因为缺乏文本的支持，你说的观点或者想法，就没有什么学术性，也就没有什么可靠性，很难说服别人。

像刚才讲信息量的例子，说李白喝的酒那么多，是有文本支持的，很多李白诗的细节都有文本支持。

例如白酒新熟山中归，黄鸡啄黎秋正肥，这首诗酒提到了白酒。

讲到这里如果你对于李白喝什么酒这个回答尽信无疑的话，事实上还是违背了研究的精神，研究的精神中有一个关键的部分，就是质疑。

# 8.好文章要有严密的逻辑论证

李白的诗中虽然提到了白酒，但据可查的史料记载，今天我们喝的蒸馏白酒是元代才开始有的，而李白生活在唐代，中间还隔了宋代，因此李白的诗中提到的白酒到底是不是白酒，我们不可以仅从字面去解读。有了论证的过程，你的表达才更严密，更有可信度。

这个例子就呈现出了信息量，文本支持，逻辑论证这三点。

# 9.给你的写作找一个导师

如果你想让你的研究更好，你可以和人讨论，给自己找一个导师。

总结：如果你的研究让你有了新的发现和收货，你应该把它写成一篇系统的，有文本支持的，逻辑论证严密的好文章。

# 10.学术论文的写作要点：

什么是可靠性？可靠性就是你说的话得有出处，在生活中我们当然可以随意的说些自己的想法，但是在学术体系中，你在论文的写作中，几乎每一句话都要有出处，不能只是你自己的想法，而要引用别人的话。

引用到什么程度呢？举个例子，比如你在你的论文里要写一句话，这句话是谁说的，哪年说的，在哪本书或者期刊资料里说的，在那本书的哪个版本哪一页说的，都要写清楚。

事实上，论文的写作，讲究的是借力打力，因为在庞大的学术体系中，任何一个人都是微不足道的，你要将学术体系的某一个微笑的分支进行那么一点点的发展，必须有扎实的根基，这个根基就是你要有大量的学术阅读，以获得一个学术的背景。

表达的时候要注意分寸，我在英国学习学术写作的时候，老师一开口就跟我们说，在学术论文中，是绝对不可以出现I这个词，就是中文里的我，你绝对不可以说“i think” 或 "i believe" 之类的话，如果你这么写，那么你的导师很可能问你：“你是谁？"

事实上，你基于大量的学术阅读，把符合或者说适合你要表达的观点的内容引用出来，也就相当于我们表达了自己的观点，这就是我说的借力打力。

# 学术论文的写作要点有：

正确性，可靠性，可持续性，普适性

# 11.为什么要会研究？

研究这件事：

职场晋升和技能考试，你要在短时间内快速做好研究，才能有针对的进行准备

给老板汇报，你需要在两天内，研究好一个新产品，或是一家竞品公司，把握核心

想给家里买台空气净化器，你的研究好哪个品牌和型号的性价比最高

想挑选一只股票，也得提前研究一下背景和行情

换句话说，一切能帮你把事情做的更好的东西，都值得你去研究。
